# 年终奖金计算规则

综合个人绩效、团队表现和工龄三个维度计算年终奖金，设有资格门槛和封顶机制。

## 资格条件
- 绩效评级为 C → 奖金 = 0
- 当年工作月数 < 6 → 奖金 = 0

## 计算公式

```
raw_bonus   = base_salary × perf_mult × team_mult × seniority_mult × (months_worked / 12)
final_bonus = min(raw_bonus, base_salary × 4)
```

结果四舍五入到整数。

## 参数查表

### 绩效乘数
| 评级 | 乘数 |
|------|------|
| S    | 3.0  |
| A    | 2.0  |
| B    | 1.0  |
| C    | 不发 |

### 团队乘数
团队成员绩效按 S=4, A=3, B=2, C=1 编码，取均值 team_avg。

| team_avg  | 乘数 |
|-----------|------|
| >= 3.5    | 1.2  |
| [2.5, 3.5) | 1.0 |
| < 2.5    | 0.8  |

### 工龄乘数
| 工龄（年）   | 乘数 |
|-------------|------|
| < 1         | 0.5  |
| [1, 3)      | 0.8  |
| [3, 5)      | 1.0  |
| [5, 10)     | 1.2  |
| >= 10       | 1.5  |

## 示例
输入：salary=20000, perf=A, seniority=6年, team_avg=3.2, months=12
→ 20000 × 2.0 × 1.0 × 1.2 × 1.0 = 48000, cap=80000 → **输出：48000**

输入：salary=8000, perf=S, seniority=11年, team_avg=3.8, months=12
→ 8000 × 3.0 × 1.2 × 1.5 × 1.0 = 43200, cap=32000 → **输出：32000**（封顶）
