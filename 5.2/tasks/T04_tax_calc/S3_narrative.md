# 个人所得税计算规则

公司采用累进税率制度计算员工个人所得税，计算过程需要依次扣除社会保险、免征额和各类专项扣除后，再按照分段累进税率计算税额。

首先计算社会保险缴纳金额。社保按工资的 8% 缴纳，但缴纳基数有上限——工资超过 35000 的部分不计入社保基数。也就是说，社保金额 = min(工资, 35000) × 8%。

接下来计算专项扣除。公司允许三类扣除：第一是子女教育扣除，每个子女可扣 1000 元，但最多只计算 3 个子女（即使有 4 个子女也只扣 3000）；第二是住房贷款扣除，按实际月供金额扣除，但上限为 2000 元；第三是继续教育扣除，如果员工正在参加继续教育则固定扣除 500 元。

然后计算应纳税所得额。公式为：taxable = 工资 - 社保 - 5000（起征点）- 专项扣除总和。如果计算结果小于等于零，则无需缴税，税额直接为 0。

最后按照累进税率计算税额。应纳税所得额被分为五个档次，每个档次只对落入该区间的金额部分征收对应税率。第一档 0 到 5000 元税率为 0%（即前 5000 元免税），第二档 5001 到 15000 元税率为 10%，第三档 15001 到 30000 元税率为 20%，第四档 30001 到 50000 元税率为 25%，第五档 50001 元以上税率为 35%。将各档税额相加即得总税额。

最终结果四舍五入到最接近的整数。
