# 个人所得税计算

公式：taxable = salary - min(salary,35000)×8% - 5000 - deductions，按累进税率计税，四舍五入到整数。

扣除项：子女 1000/人（最多3人）、房贷（实际额，上限2000）、继续教育 500（固定）。taxable ≤ 0 则 tax=0。

累进税率：0-5000→0%, 5001-15000→10%, 15001-30000→20%, 30001-50000→25%, 50001+→35%。各档仅对落入该区间的部分征税。

## 示例 1：仅基本扣除
输入：salary=20000, children=0, mortgage=0, education=false
计算：SI = min(20000,35000)×0.08 = 1600, ded=0, taxable=20000-1600-5000-0=13400
税额：5000×0% + 8400×10% = 840
**输出：840**

## 示例 2：多项扣除 + 高收入
输入：salary=60000, children=2, mortgage=1800, education=true
计算：SI = min(60000,35000)×0.08 = 2800, ded=2000+1800+500=4300, taxable=60000-2800-5000-4300=47900
税额：5000×0% + 10000×10% + 15000×20% + 17900×25% = 0+1000+3000+4475 = 8475
**输出：8475**

## 示例 3：扣除后 taxable ≤ 0
输入：salary=5500, children=1, mortgage=500, education=false
计算：SI = 5500×0.08 = 440, ded=1000+500+0=1500, taxable=5500-440-5000-1500=-1440
**输出：0**（taxable ≤ 0，无需缴税）

## 示例 4：子女数超限 + 房贷超限
输入：salary=80000, children=5, mortgage=5000, education=true
计算：SI = 35000×0.08 = 2800（社保基数封顶）, ded=3000（子女封顶3人）+2000（房贷封顶）+500=5500
taxable=80000-2800-5000-5500=66700
税额：0 + 1000 + 3000 + 5000 + 5845 = 14845
**输出：14845**

## 示例 5：社保基数封顶
输入：salary=45000, children=0, mortgage=0, education=false
计算：SI = 35000×0.08 = 2800, ded=0, taxable=45000-2800-5000=37200
税额：0 + 1000 + 3000 + 1800 = 5800
**输出：5800**
