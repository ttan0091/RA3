# 学生成绩评定规则

学校的成绩评定系统综合考虑多项成绩和出勤情况，同时设有荣誉表彰和学业预警两种机制。

成绩的计算首先需要将四项成绩按权重加总。作业占 20%，期中考试占 25%，期末考试占 35%（权重最大，体现对终考的重视），项目占 20%。即 weighted = homework × 0.20 + midterm × 0.25 + final × 0.35 + project × 0.20。

出勤率对成绩有重大影响。如果出勤率低于 60%，学生将被直接判定为 F，无论加权分数有多高——这是一条强制规则，覆盖所有其他计算。出勤率在 60% 到 80% 之间（含 60%，不含 80%）的学生，其加权分数会被打九折（乘以 0.9），反映出勤不足的惩罚。出勤率达到 80% 及以上则不受影响。

等级根据调整后的分数判定。90 分及以上为 A，80 分及以上为 B，70 分及以上为 C，60 分及以上为 D，低于 60 分为 F。

荣誉表彰的条件相当严格：必须获得 A 等级，同时四项单科成绩必须全部达到 85 分及以上。也就是说，即使加权分达到了 A，但如果有任何一科低于 85 分，也无法获得荣誉。

学业预警针对的是成绩较差且存在明显弱项的学生。具体条件为：等级为 D 或 F，且四项成绩中有任意一项低于 50 分。注意"低于 50"是严格小于，50 分本身不触发预警。
