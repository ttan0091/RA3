# 个人所得税计算规则

采用累进税率制度，先扣除社保、免征额和专项扣除，再分段计税。

## 社会保险
- 缴纳基数 = min(salary, 35000)
- social_insurance = 基数 × 8%

## 专项扣除

| 类别     | 计算方式                          |
|----------|-----------------------------------|
| 子女教育 | 每人 1000，最多 3 人              |
| 住房贷款 | 实际金额，上限 2000               |
| 继续教育 | 有则 500（固定）                  |

## 应纳税所得额

```
taxable = salary - social_insurance - 5000 - deductions
```
若 taxable ≤ 0，则 tax = 0。

## 累进税率

| 应纳税所得额区间 | 税率 |
|-----------------|------|
| 0 ~ 5000        | 0%   |
| 5001 ~ 15000    | 10%  |
| 15001 ~ 30000   | 20%  |
| 30001 ~ 50000   | 25%  |
| 50001+          | 35%  |

各档仅对落入该区间的部分征税（累进制）。

## 示例
输入：salary=20000, children=0, mortgage=0, education=false
→ SI=1600, ded=0, taxable=13400, 税额=5000×0%+8400×10%=840 → **输出：840**

输入：salary=60000, children=2, mortgage=1800, education=true
→ SI=2800, ded=4300, taxable=47900, 税额=0+1000+3000+4475=8475 → **输出：8475**

结果四舍五入到整数。
