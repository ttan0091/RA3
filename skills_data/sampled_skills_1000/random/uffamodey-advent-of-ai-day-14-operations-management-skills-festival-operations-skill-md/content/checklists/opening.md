# Festival Opening Checklist

## Pre-Opening (Day Before)
- [ ] Confirm all vendor arrival times
- [ ] Verify security personnel schedules
- [ ] Test all sound and lighting systems
- [ ] Inspect safety equipment (barriers, fire extinguishers, first aid stations)
- [ ] Brief all staff on emergency procedures
- [ ] Verify weather contingency plans
- [ ] Final walkthrough with local authorities

## Opening Day Setup (4-6 hours before gates open)
- [ ] Set up entrance/exit areas and signage
- [ ] Deploy security checkpoints
- [ ] Activate vendor booths and food stations
- [ ] Test communication systems (radios, PA system)
- [ ] Position crowd control barriers
- [ ] Set up information booths
- [ ] Verify restroom facilities are stocked and clean

## Gates Open Preparation (1-2 hours before)
- [ ] Final sound check for all stages
- [ ] Staff briefing and position assignments
- [ ] Emergency services on standby
- [ ] Cash handling procedures reviewed
- [ ] Weather monitoring active
- [ ] Media/press area prepared

## Opening Ceremony
- [ ] Welcome announcement
- [ ] Safety reminders to attendees
- [ ] Schedule announcements
- [ ] Official opening declaration

## Post-Opening Monitoring (First 2 hours)
- [ ] Monitor crowd flow and capacity
- [ ] Check vendor operations
- [ ] Verify emergency exits remain clear
- [ ] Communication systems functioning
- [ ] Document any immediate issues
