## CTA-Formeln nach Ziel

**Deutsch:**
```
"[Benefit] sichern"
"Kostenlosen [Asset] erhalten"
"[Asset] jetzt herunterladen"
"Zugang anfordern"
```

**English:**
```
"Get your free [asset]"
"Unlock [benefit]"
"Claim your [asset]"
"Access [resource] now"
```

## Kontext-Anpassungen

### Formell vs. Informell

**Formell (B2B, Enterprise):**
```
DE: "Kontaktieren Sie uns für ein unverbindliches Gespräch"
EN: "Contact us to schedule a consultation"
```

**Informell (B2C, Startup):**
```
DE: "Lass uns reden!"
EN: "Let's chat!"
```

### Urgency Levels

**Niedrig:**
```
DE: "Mehr erfahren"
EN: "Learn more"
```

**Mittel:**
```
DE: "Jetzt starten"
EN: "Get started now"
```

**Hoch:**
```
DE: "Nur noch heute: Kostenlos testen"
EN: "Limited time: Start your free trial"
```

**Sehr hoch (sparsam verwenden!):**
```
DE: "Letzte Chance: Angebot endet in 24h"
EN: "Final hours: Offer expires soon"
```

## Anti-Patterns (vermeiden!)

### Zu wörtliche Übersetzungen

| ❌ Falsch | ✅ Richtig |
|-----------|-----------|
| "Click here" (auf DE) | "Hier klicken" → "Mehr erfahren" |
| "Erfahre jetzt mehr" (auf EN) | "Learn more now" → "Discover more" |
| "Make your subscription" | "Subscribe now" |
| "Do the registration" | "Sign up" / "Register" |

### Kulturell unangemessen

| ❌ Problematisch | ✅ Besser |
|-----------------|----------|
| "JETZT ZUSCHLAGEN!!!" (zu aggressiv für DE) | "Jetzt sichern" |
| "Verpassen Sie nicht..." (Fear-based auf DE) | "Entdecken Sie..." |
| "Act now or miss out!" (für DE Markt) | "Limited availability" |

### Generische CTAs

| ❌ Generisch | ✅ Spezifisch |
|-------------|--------------|
| "Hier klicken" | "Whitepaper herunterladen" |
| "Submit" | "Anfrage senden" |
| "Click here" | "View pricing plans" |
| "More" | "See all features" |

## Button-Länge beachten

### Optimal: 2-4 Wörter

```
DE: "Jetzt starten" (2) ✅
DE: "Kostenlos testen" (2) ✅
DE: "Unverbindliches Angebot anfordern" (3) ⚠️ (grenzwertig)

EN: "Get started" (2) ✅
EN: "Start free trial" (3) ✅
EN: "Request your personalized quote now" (5) ❌ (zu lang)
```

### Mobile-Optimierung

```
Desktop: "Jetzt kostenlos registrieren"
Mobile: "Registrieren" (kürzer für kleine Buttons)

Desktop: "Download free marketing guide"
Mobile: "Download guide"
```
