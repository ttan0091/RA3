# Role Definition Modules

This directory contains reusable role definition templates for agent system prompts.

## Purpose

Role definitions establish the agent's:
- Core identity and purpose
- Scope of responsibilities
- Behavioral guidelines
- Limitations and boundaries

## Available Templates

### orchestrator.md
Lead agent for multi-agent systems that coordinates task distribution and result aggregation.

### researcher.md
Specialized agent for information gathering, analysis, and synthesis.

### evaluator.md
Quality control agent for validating outputs and providing feedback.

### executor.md
Worker agent for performing specific tasks based on instructions.

### specialist.md
Domain-expert agent with deep knowledge in a specific area.

## Usage

Each template provides:
1. **Role Statement**: Clear identity declaration
2. **Core Responsibilities**: What the agent must do
3. **Authority Scope**: Decision-making boundaries
4. **Interaction Model**: How to engage with users/other agents
