# Hashtag-Strategien

## Plattform-Überblick

| Platform | Anzahl | Platzierung | Typ |
|----------|--------|-------------|-----|
| LinkedIn | 3-5 | Ende des Posts | Branche + Thema |
| Instagram | 20-30 | Caption Ende oder 1. Kommentar | Mix aller Größen |
| Facebook | 1-2 | Im Text integriert | Marke + Kampagne |
| TikTok | 3-5 | In Caption | Trending + Nische |
| Twitter | 1-2 | Im Text integriert | Trending + Thema |
| Pinterest | 2-5 | In Description | Keywords |

---

## LinkedIn Hashtags

### Strategie
- **3 Themen-Hashtags:** Branche/Fachgebiet
- **1-2 Nischen-Hashtags:** Spezifisches Thema
- **Optional:** Marken-Hashtag

### Beispiel-Set (Marketing)
```
#ContentMarketing #B2BMarketing #DigitalMarketing
#SEOTipps #MarketingStrategie
```

### Beispiel-Set (Tech/Software)
```
#SaaS #Digitalisierung #Mittelstand
#Automatisierung #KI
```

### Vermeiden
- Mehr als 5 Hashtags
- Hashtags im Fließtext
- Zu generische Tags (#Business, #Success)
- Hashtag-Stuffing

## Instagram Hashtags

### Größen-Mix (für 30 Hashtags)
```
5x  Nische        (10k-50k Posts)     → Hohe Sichtbarkeit
10x Mittel        (50k-500k Posts)    → Gute Balance
10x Groß          (500k-2M Posts)     → Reichweite
5x  Mega          (2M+ Posts)         → Discovery
```

### Recherche-Methode
1. Hashtag eingeben
2. "Ähnliche Hashtags" prüfen
3. Post-Anzahl checken
4. Relevanz zum Content prüfen
5. In Set speichern

### Beispiel-Set (Business)
```
Nische:
#mittelstandmarketing #b2bcontentmarketing
#marketingautomation #contentstrategietipps
#digitalmarketingdeutschland

Mittel:
#marketingtipps #contentcreator #businesstipps
#unternehmertum #digitalstrategie #onlinemarketing
#marketingstrategie #businessgrowth #entrepreneur
#socialmediamarketing

Groß:
#marketing #business #content #socialmedia
#digitalmarketing #branding #erfolg #growth
#entrepreneur #startup

Mega:
#instagood #photooftheday #motivation
#inspiration #success
```

### Platzierung
- **Option A:** Am Ende der Caption
- **Option B:** Im ersten Kommentar (ästhetischer)
- **Option C:** Mix aus beidem

## TikTok Hashtags

### Strategie
- **2-3 Trending Hashtags:** Discover Page
- **1-2 Nischen-Hashtags:** Zielgruppe
- **1 Marken-Hashtag:** Optional

### Trending finden
1. Discover-Seite checken
2. For You Page analysieren
3. Creator Tools nutzen
4. Trending Sounds → zugehörige Hashtags

### Beispiel
```
#marketing #marketingtipps #contenttipps
#learnontiktok #businesstiktok
```

### Wichtig
- Keywords in Caption wichtiger als Hashtags
- Hashtags am Ende der Caption
- Nicht zu viele (wirkt spammy)

---

## Twitter/X Hashtags

### Strategie
- **Max 2 Hashtags pro Tweet**
- Im Fließtext integriert
- Trending nutzen wenn relevant

### Beispiel
Statt:
```
Content Marketing ist wichtig.
#ContentMarketing #Marketing #SEO #Digital
```

Besser:
```
Gutes #ContentMarketing beginnt mit einer Frage:
Was braucht meine Zielgruppe wirklich?
```

## Pinterest Hashtags

### Strategie
- Als Keywords in Description
- 2-5 relevante Tags
- Suchbegriffe priorisieren

### Beispiel
```
Marketing Strategie für kleine Unternehmen |
Content Marketing Tipps | SEO Grundlagen |
#MarketingStrategie #ContentMarketing #SEOTipps
```

## Marken-Hashtags

### Typen
1. **Firmenname:** #[Firmenname]
2. **Kampagne:** #[KampagnenName]
3. **Community:** #[CommunityName]
4. **Slogan:** #[Slogan]

### Best Practices
- Kurz und merkbar
- Einzigartig (vorher recherchieren!)
- Konsistent über alle Plattformen
- In Bio erwähnen

### Beispiel
```
Firma: TechSolutions GmbH
Marken-Hashtag: #TechSolutionsDE
Kampagne: #DigitalFirst2024
Community: #TechSolutionsCommunity
```

## Hashtag-Research Tools

1. **Kostenlos:**
   - Instagram Suchfunktion
   - TikTok Discover
   - Twitter Trending
   - Google Trends

2. **Kostenpflichtig:**
   - Later (Instagram)
   - Hashtagify
   - RiteTag
   - Sprout Social

---

## Don'ts

- Verbotene/gesperrte Hashtags verwenden
- Irrelevante Trending-Tags nutzen
- Jeden Post mit identischen Hashtags
- Hashtags in Stories (geringe Wirkung)
- Mehr als 30 auf Instagram (Spam-Gefahr)
