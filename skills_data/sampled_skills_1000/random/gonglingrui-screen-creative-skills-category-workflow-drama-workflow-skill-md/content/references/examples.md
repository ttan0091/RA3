# 情节点戏剧功能分析工作流编排专家 - 详细示例

## 说明

本技能协调整个情节点戏剧功能分析流程，管理文本预处理、并行分析、结果整合及报告生成。

## 示例 1: 职场冲突分析

### 输入

```
请对以下故事进行戏剧功能分析：

林浅在会议上公开质疑上司的错误决策。上司愤怒，试图压制她。但林浅坚持立场，用数据和事实说话。最终，上司的方案被否决，林浅的方案获得采纳。
```

### 输出

```
【情节点戏剧功能分析报告】

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
一、分析概览
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 文本长度：87字
- 分析片段数：1
- 情节点总数：1

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
二、情节点分析
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【情节点】：林浅在会议上公开质疑上司的错误决策

【戏剧功能】：
该情节点是故事的核心冲突点，通过职场对抗展现了主角的正直和勇气。冲突源于真理与权力的对抗，在公开场合爆发，张力强烈。主角通过专业能力赢得胜利，实现情绪释放。

【情节影响】：
- 对主角：确立林浅正直勇敢、专业能力突出的人物形象
- 对情节：为后续林浅被迫离职埋下伏笔，推动故事发展
- 对主题：展现真理与权力的对抗，弘扬职业操守

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
三、戏剧结构分析
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【冲突类型】：上下级权力冲突 + 真理与利益冲突

【戏剧张力曲线】：
压抑阶段：上司的压制，观众同情主角
对抗阶段：林浅用数据和事实坚持立场
释放阶段：方案被否决，林浅的方案获得采纳

【情绪设计】：
- 压抑感：上司的愤怒和压制让观众产生压抑
- 期待感：观众期待林浅能够坚持
- 满足感：林浅成功坚持，获得胜利，观众获得满足

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
四、专业洞察
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
从编剧角度分析：
1. 冲突设置：选择公开会议作为冲突场景，增强戏剧张力
2. 人物对比：林浅的专业理性与上司的专断保守形成鲜明对比
3. 胜利方式：通过数据和专业能力获胜，而非情感宣泄，更具说服力
4. 后续铺垫：这次胜利为后续上司的报复埋下伏笔，符合戏剧逻辑

优化建议：
- 可增加会议规模和重要性的描述，提升风险
- 可增加同事的沉默或窃窃私语，强化孤立感
- 可让上司的威胁更具体，增加后续风险
```

## 示例 2: 情感告白场景分析

### 输入

```
请对以下情节点进行戏剧功能分析：

男主在雨夜向女主告白。前女友突然出现，打断告白。男主明确拒绝前女友。女主感动，确认关系。
```

【情节点戏剧功能分析报告】

一、分析概览
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 文本长度：46字
- 分析片段数：1
- 情节点总数：4

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
二、情节点分析
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【情节点1】：男主在雨夜向女主告白
【戏剧功能】：营造浪漫氛围，建立情感期待

【情节点2】：前女友突然出现，打断告白
【戏剧功能】：制造冲突和悬念，升级戏剧张力

【情节点3】：男主明确拒绝前女友
【戏剧功能】：释放观众情绪，展现男主坚定态度

【情节点4】：女主感动，确认关系
【戏剧功能】：实现情感满足，完成故事闭环

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【情节点序列】：告白→被打断→拒绝前任→确认关系

【情绪曲线设计】：
浪漫期待（告白）→ 紧张冲突（前任出现）→ 爽快释放（拒绝前任）→ 温馨满足（确认关系）
整体呈现"浪漫-紧张-爽快-温馨"的完美情绪闭环

【戏剧张力层次】：
- 第一层：告白时的紧张期待（她会接受吗？）
- 第二层：前任打断的冲突升级（会发生什么？）
- 第三层：拒绝前任的爽快释放（他选择了女主）
- 第四层：确认关系的温暖收尾（终成眷属）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
从编剧角度分析：
1. 节奏把控：四个情节点层层递进，情绪起伏恰到好处
2. 冲突设计：前任的出现是经典但有效的冲突设置
3. 情绪释放：拒绝前任的情节提供了爽感释放点
4. 场景利用：雨夜场景增强了浪漫氛围和视觉冲击

- 男主→女主：从暧昧到明朗，关系得到确认

- 女主：从被动等待到主动接受，完成角色成长

优化建议：
- 可让告白的台词更加具体和有针对性
- 可让前任的出现方式更具戏剧性（如：不仅是出现，还有具体的行动）
- 可增加女主的内心活动和反应，增强共情
```

## 示例 3: 长文本分析

### 输入

```
请对以下故事进行戏剧功能分析：

苏婉是礼部尚书的嫡女，聪慧过人。她入宫选秀，被皇上选中。入宫后，苏婉发现后宫中充满了勾心斗角。苏婉凭借智慧和谨慎，在后宫站稳脚跟。

皇后是皇上在潜邸时的发妻，地位尊贵，但多年无子让她倍感威胁。她视苏婉为眼中钉，设计陷害苏婉在皇上面前失仪。苏婉被打入冷宫。

在冷宫的日子里，苏婉心灰意冷，但一次偶然的机会，她结识了被废的太妃。太妃曾经也是后宫的宠妃，因被皇后陷害而失去一切。太妃看到苏婉的遭遇，对她心生怜悯，决定教导她宫廷斗争的技巧。

在太妃的指导下，苏婉开始改变自己，从单纯的少女逐渐变得深不可测。她学会了隐忍和等待，学会了在后宫的复杂环境中保护自己。

终于，机会来了。皇上出巡途中遭遇刺客袭击，随行的侍卫死伤惨重。苏婉冒险救驾，用自己的身体为皇上挡下了致命一击。她的忠诚和勇敢感动了皇上，也让她重新获得了皇上的宠爱。

回到宫中后，苏婉不再像以前那样单纯。她开始在暗中收集皇后陷害她的证据，同时积极拉拢其他妃嫔，建立自己的势力。经过周密的布局，苏婉在朝廷上公开揭露了皇后的阴谋。证据确凿，皇后被打入冷宫，苏婉成功上位，成为了新的皇后。
```

3. 冲突设置：与皇后的对抗贯穿始终，矛盾主线明确

- 成长与复仇：从受害者到复仇者的成长历程

- 可在冷宫阶段增加更多内心戏，展现人物挣扎
- 可让太妃这个角色更加立体，增加师徒情谊的描写
- 可在最终胜利时增加反思，增强主题深度
```
